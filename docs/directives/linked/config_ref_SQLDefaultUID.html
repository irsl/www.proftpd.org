SQLDefaultUID

Name

SQLDefaultUID -- Configure the default UID for users

Synopsis

SQLDefaultUID [ defaultuid]

Default

65533

Context

server config, <Global>, <VirtualHost>

Module

mod_sql

Compatibility

1.2.0 and later

Description

Sets the default UID for users. Must be greater than SQLMinID.

See also

SQLMinID SQLMinUserUID
